## 🎯 Purpose
This portfolio serves as a central hub for my projects, design approach, and professional background.  
It highlights my strengths in UI/UX design, frontend development, and building polished user interfaces.

---

## 🛠 Tech Stack
- **HTML5**, **CSS3**, **JavaScript**
- **Alpine.js** — State management
- **Canvas API** — Wave animations

---

## 🧩 Features
- Modern, minimalist design with animated wave background (Canvas)
- Light/Dark theme toggle with localStorage persistence
- Dynamic wave animation that adapts to theme colors
- Monospaced font option
- Fully responsive layout
- Project showcase
- Contact form
